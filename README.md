# Blinkit Sales Dashboard (Power BI)

##  Overview

This project is an interactive Power BI dashboard built to analyze sales performance, outlet characteristics, and product distribution for a retail dataset (Blinkit-style grocery data).

The dashboard provides insights into sales trends, item categories, outlet performance, and customer preferences through dynamic visualizations and filters.

## Key Features

*  Total Sales, Average Sales, and Item Count KPIs
*  Outlet analysis by size, location, and type
*  Product insights by item type and fat content
*  Year-wise outlet establishment trend
*  Average rating analysis across outlets
*  Interactive slicers for filtering:

  * Outlet Location
  * Outlet Size
  * Item Type

---

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)

---

## Business Insights

* Identifies top-performing item categories
* Compares sales across outlet tiers
* Helps understand customer preferences (low fat vs regular)
* Tracks growth trends over time
